# Example d'application symfony

Cette application est un exemple d'application de l'article https://gkueny.fr/symfony-pouvoir-se-logguer-avec-deux-entite-differentes

## Installation

```shell
composer install
```

## Lancer le serveur

```shell
symfony serve
```
